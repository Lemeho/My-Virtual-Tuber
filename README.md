# MVt

## Русский/Russian

My Virtual tuber или MVt это простая программа для создания виртуального аватара.
Мне было лень разбираться в теме прог для втуберства и интереса ради решил сделать свою.

### Как с ней работать?

- Для начала, в настройках следует выбрать корневую папку. Там будут храниться все аватары.
- Создать аватара
- Добавить состояние
- Настроить состояние
- Повторить по необходимости

Функционал будет дорабатываться. 


## English/Английский

My Virtual tuber or MVt is a simple program for creating a virtual avatar.
I was too lazy to understand the topic of Vtuber progs and for the sake of interest I decided to make my own.

### How to work with it?

#### (Default language is Russian. If you want to change it, click top-right button for settings)

- First, in the settings you should select the root folder. All avatars will be stored there.
- Create an avatar
- Add state
- Customize the state
- Repeat as necessary

The functionality will be improved.
